## Instalación

1. Descargar el repositorio.
2. Copiar la carpeta Hospital en htdocs (XAMPP) o www (WAMP).
3. Importar el archivo pacientes.sql en phpMyAdmin.
4. Configurar conexion.php.
5. Abrir en el navegador:

http://localhost/Hospital

- Registrar pacientes
- Mostrar pacientes
